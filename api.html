מסדירים

חוזרים לחלוקת התנ"ך היהודית





החלוקה לתנ"ך לסדרים זמינה גם דרך אתר מסדירים www.masdirim.org באמצעות הממשק למשתמש,  וניתן לגשת למידע גם באמצעות API וקבצי JSON. קובץ JSON המלא המכיל את כל פסוקי התנ"ך זמין להורדה בכתובת:
https://raw.githubusercontent.com/bambiker/sdarim/refs/heads/main/sdarim.json
ניתן דרך API לגשת לכל פסוק בנפרד לפי החלוקה לפרקים בכתובת:
https://raw.githubusercontent.com/bambiker/sdarim/refs/heads/main/API/chapter_פסוק_פרק_ספר.json
לדוגמה, עבור ספר דברי הימים א', פרק ב', פסוק ג' (לפי החלוקה לפרקים)
https://raw.githubusercontent.com/bambiker/sdarim/refs/heads/main/API/chapter_ג_ב_דברי_הימים_א.json
בנוסף, ניתן לגשת גם לפי החלוקה לסדרים בכתובת
https://raw.githubusercontent.com/bambiker/sdarim/refs/heads/main/API/seder_פסוק_פרק_סדר.json
למשל, עבור ספר דברי הימים, סדר א', פסוק נז' (לפי החלוקה לסדרים)
https://raw.githubusercontent.com/bambiker/sdarim/refs/heads/main/API/seder_נז_א_דברי_הימים.json
מבנה התשובה של ה-API יהיה כך:

    "chapter": "ב",
    "versechapter": "ג",
    "bookseder": "דברי הימים",
    "seder": "א",
    "verseseder": "נז",
    "versenikud": "בְּנֵ֣י יְהוּדָ֗ה עֵ֤ר וְאוֹנָן֙ וְשֵׁלָ֔ה שְׁלוֹשָׁה֙ נ֣וֹלַד ל֔וֹ מִבַּת־ שׁ֖וּעַ הַֽכְּנַעֲנִ֑ית וַיְהִ֞י עֵ֣ר ׀ בְּכ֣וֹר יְהוּדָ֗ה רַ֛ע בְּעֵינֵ֥י יְהוָ֖ה וַיְמִיתֵֽהוּ׃ ",
    "versenonikud": "בני יהודה ער ואונן ושלה שלושה נולד לו מבת שוע הכנענית ויהי ער  בכור יהודה רע בעיני יהוה וימיתהו "
}

רשימת הספרים נמצאת בכתובת
https://raw.githubusercontent.com/bambiker/sdarim/refs/heads/main/API/book_names.json
